Завдання 4. Багатокритерійна оптимізація

Для заданих альтернатив і заданих значень критеріїв виконати ранжирування з урахуванням вагових коефіцієнтів (альтернативи, значення критеріїв і вагові коефіцієнти задаються довільно).

Для цього необхідно:
Нормалізувати критерії і вагу критеріїв.
Проранжирувати альтернативи по суперкритеріям:
a)аддитивному
b)мультиплікативному
c)Кобба - Дугласа
Застосувати метод головного приватного критерія, вважаючи, що по важливості критерії розташовані по зменшеню F1,F2,F3…
Проранжирувати альтернативи по методу ідеальної точки, використовуючи евклідову міру k=2
Виконати ранжирування по Парето, використовуючи таблицю переваг
